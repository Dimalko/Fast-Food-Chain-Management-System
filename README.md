## Overview
The **Fast Food Chain Management System** is a web-based CRUD application developed using **PHP** and **MySQL**. It is designed to help manage the daily operations of a fast food business efficiently.

The system provides an easy way to organize and maintain data related to:
- Meals  
- Employees  
- Orders  
- Ingredients  

---

## Features
- Full **CRUD Operations** (Create, Read, Update, Delete)  
- Modular and organized PHP structure  
- Relational database using MySQL  
- Simple, clean, and user-friendly interface  

---

## Technologies Used
- **PHP**  
- **MySQL**  
- **HTML5**  
- **CSS3**  

---

## Setup Instructions

### 1. Clone the Repository
```bash
git clone https://github.com/Dimalko/fast-food-management-system.git
```
### 2.Move Project to Server Directory

Place the project folder inside your local server directory (e.g., htdocs for XAMPP).

### 3. Import the Database
- **Open phpMyAdmin**
- **Create a new database (optional)**
- **Import the file:**
```bash
fast_food_chain.sql
```

### 4. Run the Project

Open your browser and navigate to:

```bash
http://localhost/fast-food-management-system
```
